# Dog Doing

一个 2D 俯视角乱斗游戏。

## 玩法简介

进入游戏后先选角色，然后直接进入竞技场开始混战。  
目前一共有三个可选角色：

- Dog Doing
- Hajimi
- Bibilabu

角色主要体现在形象和战斗气质上，进场后都能立刻参与战斗。

## 操作

- `WASD`：移动
- `J`：攻击
- `K`：格挡

## 战斗规则

- 每个玩家出生时默认拿着小刀。
- 场地里会刷新地面武器，靠近后会自动拾取。
- 小刀是近战武器，适合贴脸压制。
- 弓箭适合中远距离拉扯，弹道带一点弧线。
- 枪是直线高速攻击，出手快，适合压制和追击。
- 格挡只能更稳定地防正面来袭的攻击，同时会明显降低移动速度。
- 被打中会掉血，血量归零后会倒下，过一小段时间重新复活。

## 对局节奏

这个游戏现在更偏持续混战，而不是回合制胜负。  
核心乐趣在于：

- 走位
- 抢武器
- 判断对手朝向
- 用格挡吃掉正面进攻
- 找到合适距离再出手

简单来说，就是一边移动、一边换武器、一边找机会把对面先打崩。
